How It Works
// under the hood
01
TF-IDF Vectorization

Each ticket is transformed into a sparse vector. Words common across all tickets get low weight; domain-specific terms get amplified — giving the model strong signal on intent.

scikit-learn n-gram support stop-word filter
02
Multiclass Classification

Logistic Regression with a softmax output layer predicts probability across all departments simultaneously. Fast, regularized, and produces calibrated confidence scores.

LogisticRegression L2 regularization softmax
03
Confidence-Based Routing

Tickets above the confidence threshold are auto-routed instantly. Below threshold, tickets escalate to human review — preventing costly misclassification in production.

threshold = 0.75 auto-route human review
